Цикл команды
	Цикл команды - это действие, треубемое для выполнения команды
	
	Основные этапы цикла:
		1. Выборка команды
		2. Формирование адреса следующей команды
		3. Декодирование команды
		4. Вычисление адресов операндов
		5. Выборка операндов
		6. Исполнение операции
		7. Запись результата

	Основные показатели вычислительной машины:
		Ёмкость памяти
		Быстродействие
		Производительность
		Надёжность
		Стоимость
		
		Быстродействие
			1. Номинальное быстродействие - скорость выполнения стандартной типовой операции
				Vном = 1 / (ТАУ СЛОЖЕНИЯ) [оп/с]
				ТАУ СЛОЖЕНИЯ - время выполнения сложения
			2. Среднее быстродействие - скорость вычислений при выполнении эталонного алгоритма (или класса алгоритмов)
				Vср = N/Tэ
				Tэ - время выполнения эталонного алгоритма
				N - количество опрераций в эталонном алгоритме
			3. Tэ - сумма(i=1,l) ТАУi ni
				l-количество типов операций
				ni - число операций i-го типа
				ТАУi - время выполнения операции i-го типа
				ni/N = qi - частота опрераций i-го типа
				(= 1/sum(i=1,l)qi)*ТАУi) [ОП/с]
